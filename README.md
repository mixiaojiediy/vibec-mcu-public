# VibeCoding MCU 智能控制系统

这是一个基于ESP32S3的智能MCU控制系统，支持远程更新、设备管理和实时通信功能。

## 🚀 核心功能

### 网络连接
- **WiFi自动连接**：设备启动时自动连接WiFi网络
- **智能配网**：WiFi连接失败时自动开启配置热点，方便用户通过手机APP进行网络配置

### 远程更新
- **OTA远程更新**：支持通过服务器远程推送固件更新
- **版本管理**：自动备份当前版本，支持版本回退
- **安全更新**：更新前自动备份，确保更新失败时可快速恢复

### 设备通信
- **MQTT通信**：支持通过MQTT协议与服务器进行双向通信
- **UDP通信**：支持局域网内UDP直连通信，实现低延迟控制
- **设备发现**：自动广播设备信息，方便APP发现和绑定设备
- **实时遥测**：支持实时上报设备状态和传感器数据

### 系统保护
- **异常监控**：实时监控系统运行状态，异常时自动上报
- **看门狗保护**：防止系统死锁，确保设备稳定运行
- **自动恢复**：支持异常情况下的自动恢复机制

## 📱 使用场景

- 智能硬件设备的远程管理和控制
- 物联网设备的固件更新和维护
- 实时数据采集和监控
- 低延迟的本地设备控制

## 🔧 主要特性

- **简单易用**：支持手机APP一键配网，无需复杂配置
- **稳定可靠**：多重保护机制，确保设备长期稳定运行
- **灵活扩展**：支持用户自定义应用程序开发
- **安全可靠**：文件完整性验证，防止恶意更新
